# Machine Learning - Linear Regression

This repository is part of my **Machine Learning Learning Journey**. It focuses on understanding and implementing different types of **Linear Regression** using Python and Scikit-learn.

The repository contains notebooks demonstrating the complete machine learning workflow, including data preprocessing, model training, prediction, evaluation, and visualization.

## 📊 Algorithms Implemented

### 1. Simple Linear Regression
Predicts a target variable using a single independent feature.

**Workflow**
- Load dataset
- Data preprocessing
- Train-Test Split
- Model training
- Prediction
- Performance evaluation
- Regression visualization

---

### 2. Multiple Linear Regression
Predicts the target using multiple independent features.

**Workflow**
- Feature selection
- Data preprocessing
- Model training
- Prediction
- Performance evaluation

---

### 3. Polynomial Regression
Captures non-linear relationships by transforming input features into polynomial features.

**Workflow**
- Polynomial feature transformation
- Model training
- Prediction
- Visualization of fitted curve
